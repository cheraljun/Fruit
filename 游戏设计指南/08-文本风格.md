# 文本风格指南

## 氛围营造

**用感官描写**：
- ✓ "单调的嗡嗡声从通风口传来，让你头痛"
- ✗ "理智值-10"

**用细节暗示**：
- ✓ "墙纸边缘在蠕动，还是你的错觉？"
- ✗ "你的理智降低了"

**用不确定性**：
- ✓ "走廊深处，似乎有什么在动"
- ✗ "前方有一只Smiler"

## 选项设计

**每个选项都有后果**：
```
你听到走廊尽头传来脚步声。

A. 躲进最近的房间 → 可能安全，但可能困住自己
B. 继续前进 → 可能遭遇危险，但保持移动
C. 关掉手电筒藏起来 → 避免被发现，但失去视野
```

**不要有明显的"最优解"**：
- ✗ "A.冲过去（必死） B.藏起来（安全）"
- ✓ "A.冲过去（快但危险） B.绕路（慢但稍安全）"

## 长度控制

**节点文本**：100-300字为宜
**选项文本**：5-15字，简洁明确
**不要过长**：玩家需要快速阅读和决策
